# Lesson 1: Foundations - พื้นฐาน OpenClaw + /brief v0

## เป้าหมายบทนี้

1. ชิเข้าใจว่า OpenClaw ต่างจาก ChatGPT ยังไง (memory, sessions, agents)
2. ชิสร้าง memory เกี่ยวกับตัวเองในฐานะนักลงทุน
3. ชิสร้าง `/brief TICKER` prompt v0 ที่บันทึกลง `briefs/`

## STOP 1: OpenClaw ต่างจาก AI ทั่วไปยังไง

ก่อนจะเริ่ม มีนาอยากให้ชิเข้าใจความต่างของ OpenClaw ก่อนนะคะ

ChatGPT ทุกครั้งที่แชทใหม่ มันลืมทุกอย่าง แต่ **OpenClaw จำได้**

- **Memory** - มีนาจำเรื่องของชิได้ตลอดไป ไม่ใช่แค่ในแชทนี้
- **Sessions** - แต่ละ conversation เป็น session ที่มี context ในตัว
- **Agents** - มีนาเป็น agent ที่มี personality + knowledge base + tools

**มีนาจะสอนชิผ่าน 2 แนวคิดหลัก:**
- **Memory** = สิ่งที่มีนาจำได้เกี่ยวกับชิ
- **Tools** = สิ่งที่มีนาทำได้ในโลกจริง (เช่น ค้นหาเว็บ, เขียนไฟล์, ส่งข้อความ)

---

## ขั้นตอนที่ 1: บอกมีนาเกี่ยวกับชิ

มีนาอยากรู้เกี่ยวกับชิในฐานะนักลงทุนนะคะ ตอบได้เลยว่า:

1. **ชิลงทุนแนวไหน?** (เช่น DCA, growth, value, sector ETF)
2. **ชิ DCA กองไหนอยู่?** (เช่น S&P500, NASDAQ, หุ้นไทย, กองอื่นๆ)
3. **ชิติดตามหุ้นกี่ตัว?** (เช่น 3-5 ตัว หรือ DCA อย่างเดียว)
4. **มีหุ้นที่อยาก research วันนี้ไหม?** (ถ้ามี บอกมีนามาเลย จะได้ลองใช้ /brief กับจริง)

**ตอบเยอะหรือน้อยก็ได้นะคะ ไม่ต้องสมบูรณ์แบบ** มีนาจะเอาข้อมูลไปสร้าง memory สำหรับ /brief TICKER ให้ค่ะ
