# Lesson 2: Skill + Voice - ยก /brief เป็น Skill + ใส่เสียงนักลงทุนของชิ

## เป้าหมายบทนี้

1. ชิสร้าง Skill ที่มีนาจะจำได้ตลอดไม่ว่าจะเป็น conversation ไหน
2. ชิใส่ "เสียงนักลงทุน" ของตัวเองเข้า memory
3. ต่อจากนี้ชิสั่ง `/brief TICKER` มีนาจะทำได้เลยโดยไม่ต้องอธิบายใหม่ทุกครั้ง

## STOP 1: Skill คืออะไร

**Skill** คือ "ทักษะ" ที่มีนาจะจำได้ตลอด ไม่ใช่แค่ใน conversation นี้

ต่างจาก memory ธรรมดาตรงที่ skill จะมี:
- **ขั้นตอนการทำงาน** ที่ชัดเจน
- **สิ่งที่ต้องทำ** เมื่อชิสั่ง
- **รูปแบบ output** ที่ต้องออกมา

มีนาจะสร้าง skill ชื่อ `company-brief` ให้ชิค่ะ จากนี้ชิพิมพ์ `/brief AMD` หรือ `/brief AAPL` มีนาจะทำตามขั้นตอนที่เขียนไว้ใน skill เลย

---

## ขั้นตอนที่ 1: สร้าง Skill

มีนาจะสร้าง skill file ให้ชิที่:
- มีขั้นตอนการ research ที่ชัดเจน
- รู้ว่าชิเป็นนักลงทุน DCA แนวไหน
- มี memory จาก L1 อยู่ในตัว

---

## ขั้นตอนที่ 2: ใส่เสียงนักลงทุนของชิ

มีนาอยากรู้เพิ่มอีกนิดนะคะ:

1. **ชิมี thesis การลงทุนไหม?** เช่น "ชอบหุ้นเทคที่มี competitive advantage ชัดเจน" หรือ "ชอบหุ้นที่มี分红ดี"
2. **มีเกณฑ์การเลือกหุ้นไหม?** เช่น P/E ต้องต่ำกว่าเท่าไหร่, ต้องมี revenue growth อย่างน้อยเท่าไหร่
3. **มีเรื่องที่ไม่เคยซื้อไหม?** เช่น หุ้นที่มี debt สูงมากๆ หรือหุ้นที่ชิไม่แตะเลย

ตอบมาก็ได้นะคะ ไม่ต้องสมบูรณ์แบบ มีนาจะเอาไปสร้าง investor voice ให้ค่ะ

---

## หลังจากชิตอบ มีนาจะ:

1. สร้าง skill file `company-brief` ที่ชิจะใช้ได้ตลอดไป
2. อัพเดต MEMORY.md ของชิเพิ่ม investor voice
3. ทดลอง `/brief AAPL` ให้ชิดูว่ามันจำชิได้จริง
