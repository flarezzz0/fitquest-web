# 🎯 โจทย์ท้าทาย: JavaScript พื้นฐาน (บทที่ 1-10)

> **กฎ:** ลองทำด้วยตัวเองก่อน ถ้าติดค่อยขอ hint

---

## 🟢 ง่าย: โจทย์สำหรับมือใหม่

### 1. Hello JavaScript!
สร้างไฟล์ `hello.html` แล้วเขียน JavaScript ที่เมื่อเปิดหน้าเว็บแล้วแสดงข้อความ "Hello JavaScript!" ที่ console

**Hint เมื่อ Error:**
- ลองเปิด DevTools (F12) → Console ดูว่ามีอะไรขึ้นมั้ย
- เช็คว่า `<script>` tag อยู่ในตำแหน่งที่ถูกต้องไหม

---

### 2. ตัวแปรแรก
ประกาศตัวแปรชื่อ `myName` และ `myAge` เก็บชื่อและอายุของคุณ แล้วแสดงผลด้วย `console.log()` ในรูปแบบ:
```
สวัสดี ฉันชื่อ [ชื่อ] อายุ [อายุ] ปี
```

---

### 3. คำนวณเลข
เขียนโค้ดที่รับเลข 2 ตัว แล้วแสดงผล: บวก, ลบ, คูณ, หาร

---

## 🟡 ปานกลาง

### 4. ตรวจสอบคะแนน
เขียนโปรแกรมที่รับคะแนน (0-100) แล้วบอกเกรด:
- 80+ = A
- 70-79 = B
- 60-69 = C
- 50-59 = D
- <50 = F

**Hint:** ลองใช้ if-else หรือ switch

---

### 5. นับถอยหลัง
เขียน loop ที่นับถอยหลังจาก 10 ถึง 1 แล้วแสดง "🎉 Happy New Year!"

---

## 🔴 ยาก (ลองทำดู)

### 6. หาคำที่ยาวที่สุด
กำหนด array: `["แมว", "ปลา", "ช้าง", "กระต่าย", "นก"]`
ให้หาคำที่ยาวที่สุดและแสดงผล

---

### 7. ตะกร้าสินค้า
สร้าง object แทนตะกร้าสินค้า ที่มี:
- รายการสินค้า (array ของ object: ชื่อ, ราคา, จำนวน)
- method คำนวณราคารวม
- method เพิ่มสินค้า
