### 🔴 สิ่งที่ขาดวิกฤตที่ควรเพิ่มก่อนเริ่มสอน
1. **try/catch/finally** — ไม่มีในหลักสูตร ควรเพิ่มหลัง Async/Await
2. **Classes & Prototypes** — ไม่มีในหลักสูตร แต่จำเป็น
3. **Closures** — concept พื้นฐานที่ควรรู้
4. **Hoisting** — concept พื้นฐานที่ควรรู้
5. **Spread/Rest Operator** — ใช้บ่อยมากในชีวิตจริง
