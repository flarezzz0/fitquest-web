# 🎯 โจทย์ท้าทาย: JavaScript ขั้นกลาง (บทที่ 21-39)

---

## 🟢 ง่าย

### 1. Template String
ใช้ Template String สร้างประโยค:
```
สวัสดี ฉันชื่อ [name] อายุ [age] ปี เรียนที่ [school]
```

---

### 2. กรอง Array
มี array: `[15, 8, 22, 3, 17, 10]`
ให้กรองเอาเฉพาะเลขที่มากกว่า 10

---

## 🟡 ปานกลาง

### 3. Fetch API
ใช้ Fetch API ดึงข้อมูลจาก `https://jsonplaceholder.typicode.com/users`
แล้วแสดงชื่อและอีเมลของผู้ใช้ทั้งหมด

---

### 4. Async/Await
เขียนฟังก์ชัน async ที่:
1. รอ 2 วินาที (ใช้ setTimeout + Promise)
2. แล้วค่อยแสดง "เสร็จแล้ว!"

---

### 5. Destructuring
กำหนด object:
```js
const student = { name: "มีนา", age: 20, grade: "A", subjects: ["JS", "Python"] }
```
ใช้ destructuring ดึง name กับ subjects ออกมา

---

## 🔴 ยาก

### 6. Todo List (Module)
สร้าง Todo List โดยแยกเป็น module:
- `todo.js` — logic จัดการ todo (add, remove, toggle)
- `app.js` — เรียกใช้

ใช้ import/export

---

### 7. Callback → Promise → Async/Await
เขียนฟังก์ชัน `readFile(name, callback)` แบบจำลอง:
1. แบบ Callback
2. แปลงเป็น Promise
3. แปลงเป็น Async/Await

**Hint:** ใช้ setTimeout จำลองการโหลด
