## 🟢 ง่าย

### 1. เปลี่ยนสีพื้นหลัง
สร้างปุ่มที่เมื่อคลิกแล้วเปลี่ยนสีพื้นหลังของหน้า

### 2. แสดง/ซ่อนข้อความ
สร้างปุ่ม toggle ที่แสดงหรือซ่อนข้อความ

---

### 3. นับจำนวนคลิก
สร้างตัวนับที่เพิ่มขึ้นทุกครั้งที่คลิกปุ่ม แสดงผลบนหน้า

## 🟡 ปานกลาง

### 4. Dynamic List
มี input + ปุ่ม ให้ผู้ใช้พิมพ์อะไรก็ได้แล้วเพิ่มเข้า list บนหน้าเว็บ

---

### 5. แบบฟอร์มลงทะเบียน
สร้างฟอร์ม: ชื่อ, อีเมล, อายุ
เมื่อ submit ให้ตรวจสอบว่า:
- ชื่อต้องไม่ว่าง
- อีเมลต้องมี @
- อายุต้องเป็นตัวเลข
แสดง Error ถ้าผิด

## 🔴 โปรเจกต์ท้าทาย

### 🏆 Final Project: เว็บไซต์ท่องเที่ยวส่วนตัว
สร้างเว็บ 1 หน้าที่มี:

1. **Responsive Menu** — hamburger menu สำหรับมือถือ
2. **Slideshow** — รูปภาพที่เปลี่ยนอัตโนมัติ
3. **Lightbox** — คลิกที่รูปเพื่อดูใหญ่
4. **Modal** — ปุ่มที่เปิด popup ข้อมูล
5. **Accordion** — หัวข้อที่กดแล้วขยายเนื้อหา
6. **Back to Top** — ปุ่มเลื่อนขึ้นบนสุด
7. Deploy ขึ้น Netlify

> 💡 นี่คือโปรเจกต์ที่เรียนจาก MilerDev JavaScript Mastery บทที่ 47-55
> ลองทำเองก่อน ถ้าติดค่อยขอ hint!

---

### Hint สำหรับโปรเจกต์
**Responsive Menu:** ใช้ media query + JavaScript toggle class
**Slideshow:** ใช้ setInterval + เลื่อน index
**Lightbox:** สร้าง div overlay ที่ซ่อนอยู่ แล้ว show เมื่อคลิก
**Modal:** เหมือน lightbox แต่แสดงข้อความ
**Accordion:** toggle class `.active` ที่ sibling element
**Back to Top:** ตรวจสอบ scrollY แล้ว show/hide ปุ่ม
