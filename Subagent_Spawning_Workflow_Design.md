# WORKFLOW: ทีม AI สร้างข่าวหุ้นด้วย Subagent Spawning

---

## 📋 Overview

เอกสารนี้อธิบาย workflow สำหรับใช้ **Subagent Spawning** ใน OpenClaw เพื่อสร้างทีม AI ที่ทำงานร่วมกันสร้าง content ข่าวหุ้นแบบอัตโนมัติ

---

## 🔹 Subagent Spawning คืออะไร?

**Subagent Spawning** คือการที่มีนา (หัวหน้าทีม) สามารถ "แตกตัวเอง" ออกไปทำงานหลายอย่างพร้อมกัน โดยแต่ละตัวจะมีบทบาทเฉพาะทาง ทำงานจบแล้วกลับมาบอกผลลัพธ์ แล้วมีนารวมให้ชิ

### เปรียบเทียบ
| บทบาท | ความหมาย |
|-------|---------|
| 👤 มีนา (หัวหน้าทีม) | ผู้ประสานงาน สั่งงานและรวมผลลัพธ์ |
| 🤖 Subagent | ลูกน้องที่ได้รับมอบหมายงานเฉพาะทาง |
| 👑 ชิ (ผู้ตรวจสอบ) | ขั้นสุดท้าย - review และตัดสินใจ |

### ข้อดีของ Subagent Spawning
- ✅ ทำงานหลายอย่างพร้อมกัน (เร็วขึ้นมาก)
- ✅ แต่ละตัวเชี่ยวชาญในเรื่องเดียว
- ✅ ชิเป็น "ผู้ตรวจสอบ" ขั้นสุดท้าย

---

## 🏗️ วิธีใช้ใน OpenClaw

### รูปแบบคำสั่ง
เมื่อชิต้องการให้ทีม AI ทำงาน สั่งงานมีนาแบบนี้:

```
"ช่วยสร้างทีม AI 3 ตัวดังนี้:
 - ตัวที่ 1: หาข่าว S&P500, Apple, Antopic
 - ตัวที่ 2: ตรวจสอบความถูกต้อง
 - ตัวที่ 3: เขียนสครีปวิดีโอสั้น"
```

มีนาจะ spawn subagent ออกไปทำงานแต่ละตัว แล้วรวมผลมาให้ชิ

## 📊 WORKFLOW: ทีม AI ทำข่าวหุ้น

```
┌─────────────────────────────────────────────────────────────┐
│              📋 WORKFLOW: ทีม AI ทำข่าวหุ้น                 │
└─────────────────────────────────────────────────────────────┘

[1] 🔍 RESEARCH AGENT
    │
    │  ดึงข่าวจากแหล่งต่างๆ ทุกวัน (เช้า-บ่าย-เย็น)
    │  → ส่งต่อ: ข้อมูลดิบ (raw data)
    ▼
[2] ✅ VERIFY AGENT
    │
    │  ตรวจสอบความถูกต้อง
    │  แยกข่าวจริง/ปลอม
    │  แปลคำศัพท์หุ้นให้เข้าใจง่าย
    │  → ส่งต่อ: ข้อมูลที่ผ่านการกรอง (verified data)
    ▼
[3a] ✍️ SCRIPT SHORT AGENT
     │
     │  เขียนสครีปวิดีโอ 1-2 นาที
     │  สไตล์ TikTok/Shorts กระชับ ดึงดูด
     │  → ส่งต่อ: สครีปสั้น (short script)
     ▼
[3b] ✍️ SCRIPT LONG AGENT
     │
     │  เขียนสครีปวิดีโอ 2-15 นาที
     │  สไตล์ YouTube ละเอียด มีความลึก
     │  → ส่งต่อ: สครีปยาว (long script)
     ▼
[4] 👤 ชิ (ผู้ตรวจสอบ)
     │
     │  Review สครีป
     │  ✅ ถ้าผ่าน → อัดวิดีโอ!
     │  ❌ ถ้าไม่ผ่าน → ส่งกลับไปแก้
     ▼
[5] 🎬 PRODUCTION
     │
     │  อัดวิดีโอ / ตัดต่อ / เผยแพร่
     ▼
```

## 🎯 รายละเอียดของแต่ละ Agent

### [1] 🔍 RESEARCH AGENT

**บทบาท:** ดึงข่าวจากแหล่งต่างๆ

**แหล่งข่าวที่ควรดึง:**
- Yahoo Finance / Google Finance
- Bloomberg / Reuters
- Thai News / ข่าวหุ้นไทย
- Twitter/X (Financial accounts)

**ข้อมูลที่ได้:**
- ราคาปิดวันนี้
- % การเปลี่ยนแปลง
- ข่าวสำคัญที่เกี่ยวข้อง
- แนวโน้ม (uptrend/downtrend/sideways)

**Output format:**
```
📊 [หุ้น]
- ราคา: $xxx
- เปลี่ยนแปลง: +x.xx% 
- ข่าว: [สรุป 2-3 ข้อ]
- แนวโน้ม: [bullish/bearish/neutral]
```

---

### [2] ✅ VERIFY AGENT

**บทบาท:** ตรวจสอบความถูกต้องของข้อมูล

**หน้าที่:**
- ตรวจสอบว่าข่าวเป็นของจริงหรือไม่ (fake news detection)
- แยกแยะว่าข่าวไหน important vs noise
- แปลคำศัพท์เทคนิคหุ้นให้เข้าใจง่าย
- ตรวจสอบความถูกต้องของตัวเลข

**ศัพท์ที่ควรแปล:**
- Bullish = มองว่าหุ้นจะขึ้น
- Bearish = มองว่าหุ้นจะลง
- Volume = ปริมาณการซื้อขาย
- Market Cap = มูลค่าตลาด
- EPS = กำไรต่อหุ้น

**Output format:**
```
✅ [หุ้น] - ผ่านการตรวจสอบ
- ข่าวจริง: ✓
- ความสำคัญ: ระดับ [สูง/กลาง/ต่ำ]
- คำอธิบายเพิ่มเติม: [ภาษาที่เข้าใจง่าย]
```

---

### [3a] ✍️ SCRIPT SHORT AGENT

**บทบาท:** เขียนสครีปวิดีโอสั้น

**รายละเอียด:**
- ความยาว: 1-2 นาที
- สไตล์: TikTok / YouTube Shorts / Reels
- โทน: กระชับ ดึงดูด ตรงไปตรงมา
- Hook: เริ่มต้นให้น่าสนใจใน 3 วินาทีแรก

**โครงสร้างสครีป:**
```
🎬 HOOK (0-5 วินาที)
"หุ้น [ชื่อหุ้น] วันนี้ [เหตุการณ์สำคัญ]!"

📢 CONTENT (30-60 วินาที)
- สรุปข่าว 2-3 ข้อ
- ตัวเลขสำคัญ
- แนวโน้มที่คาดการณ์

🎯 CTA (5-10 วินาที)
"ถ้าชอบ content แบบนี้ กดไลค์ + ติดตามด้วยนะครับ"
```

---

### [3b] ✍️ SCRIPT LONG AGENT

**บทบาท:** เขียนสครีปวิดีโอยาว

**รายละเอียด:**
- ความยาว: 2-15 นาที
- สไตล์: YouTube ละเอียด มีความลึก
- โทน: ให้ความรู้ วิเคราะห์ มีสาระ

**โครงสร้างสครีป:**
```
🎬 INTRO (30 วินาที)
- แนะนำหุ้นที่จะพูดถึง
- สรุปสิ่งที่จะได้เรียนรู้

📊 MAIN CONTENT (5-10 นาที)
- ภาพรวมตลาดวันนี้
- วิเคราะห์เชิงลึกแต่ละหุ้น
- ปัจจัยที่ส่งผลต่อราคา
- เปรียบเทียบกับวันก่อนหน้า

📈 CONCLUSION (1-2 นาที)
- สรุปความน่าจะเป็นแนวโน้มต่อไป
- เตือนความเสี่ยง (not financial advice)

❓ Q&A (ถ้ามีเวลา)
- คำถามที่น่าสนใจจาก comments
```

## 🛠️ External Tools ที่แนะนำ

### Notion 📓
- เก็บสครีปและข้อมูลเป็นระเบียบ
- สร้าง database ของหุ้นที่ติดตาม
- Template สครีปมาตรฐาน

### Google Sheets 📊
- ตารางข้อมูลราคาหุ้น
- Track ผลตอบแทน
- สรุปข่าวรายวัน/รายสัปดาห์

### Make.com / Zapier 🔄
- รันอัตโนมัติทุกวัน
- ดึงข่าวจาก RSS / API
- ส่ง notification เมื่อมีข่าวสำคัญ

### Obsidian 🧠
- ฐานความรู้ส่วนตัว
- วิเคราะห์หุ้นแบบโน้ตละเอียด
- Graph view เชื่อมโยงหุ้นที่สัมพันธ์กัน
